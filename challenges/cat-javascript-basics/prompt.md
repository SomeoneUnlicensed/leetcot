## Кот JavaScript: Основы 🐱

JavaScript - основной язык веб-разработки. Давайте создадим кошачьи функции.

### Основы JavaScript

1. **Функции**: `function` или `const func = () => {}`
2. **Объекты**: `const cat = { name: "Барсик" }`
3. **Методы**: функции внутри объектов

### Задача

Создайте объект `Cat` с:
- Свойством `name` (string)
- Методом `meow()`, возвращающим "Мяу, {name}!"
- Методом `greet(other)`, возвращающим "{name} говорит привет {other}!"

### Пример

```javascript
const cat = new Cat("Барсик");
console.log(cat.meow()); // "Мяу, Барсик!"
console.log(cat.greet("Пушок")); // "Барсик говорит привет Пушок!"
```
