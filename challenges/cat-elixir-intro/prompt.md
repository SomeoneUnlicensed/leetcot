## Функциональный котик на Elixir 🐱⚡

Добро пожаловать в Elixir - функциональный язык программирования, работающий на виртуальной машине Erlang! Сегодня мы создадим функции для работы с котами.

### Основы Elixir

Elixir известен своим:
1. **Функциональное программирование**: immutability и pure functions.
2. **Pattern Matching**: мощная система для разбора данных.
3. **Pipelines**: элегантная композиция функций.
4. **Concurrency**: встроенная поддержка параллелизма.

### Задание:

Создай модуль `Cat` с функциями:
1. `new(name, age)`: создает новую карту кота с ключами `:name` и `:age`.
2. `meow(cat)`: возвращает "Мяу! Я {name}!"
3. `get_age(cat)`: возвращает возраст кота.
4. `grow_up(cat)`: увеличивает возраст кота на 1 и возвращает обновленную карту.

Используй карты (maps) для хранения данных кота.
