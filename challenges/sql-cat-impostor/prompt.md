## Кот-самозванец 🕵️🐱

В систему управления складом ЛитКота закрался мошенник. Кто-то добавил котов с одинаковыми именами, чтобы списывать двойные порции рыбы.

Найдите все **дублирующиеся имена** котов и подсчитайте сколько раз каждое встречается.

- `cats` (id, name, registered_at)

-- Ожидаемые столбцы: name | count
