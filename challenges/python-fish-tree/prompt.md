## Высота Дерева 🌳🐈

Мяу! Кот залез на бинарное дерево (не спрашивай как). Нам нужно узнать максимальную глубину этого дерева, чтобы понять, насколько высоко наш пушистый друг.

### Задание:
Реализуй функцию `max_depth(root)`, которая принимает корень бинарного дерева и возвращает его максимальную глубину.

Узел дерева определен так:
```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
```

```python
# Пример
#   3
#  / \
# 9  20
#    / \
#   15  7
# Результат: 3
```
