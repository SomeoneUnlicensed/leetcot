## Высота Дерева (Maximum Depth of Binary Tree) 🌳🐈

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **Apple**, **Yandex**, **Сбер**, **Т-Банк**.
>
> 😺 **ЛитКот напоминает:** Задача отлично решается через рекурсивный обход в глубину (DFS) со сложностью $O(N)$, где $N$ — число узлов в дереве.

Пушистый друг залез на бинарное дерево. Вам необходимо узнать его максимальную глубину, чтобы понять, насколько высоко он находится.

Максимальная глубина — это количество узлов на самом длинном пути от корневого узла до самого дальнего листового узла.

### Задание:

Реализуйте функцию `max_depth(root)`, которая принимает корень бинарного дерева и возвращает его максимальную глубину.

Узел дерева определен следующим образом:

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right
```

```python
# Пример
#     3
#    / \
#   9  20
#      / \
#     15  7
# Результат: 3
```
