## Кот PHP: Веб для котов 🌐

PHP - язык для веб-разработки. Давайте создадим кошачий класс для веба.

### Основы PHP

1. **Классы и объекты**: ООП в PHP
2. **Методы**: Функции внутри класса
3. **Строки**: Работа с текстом

### Задача

Создайте класс `Cat` с:
- Конструктором `__construct($name)`
- Методом `meow()`, возвращающим "Мяу, {name}!"
- Методом `greet($other)`, возвращающим "{name} говорит привет {other}!"
