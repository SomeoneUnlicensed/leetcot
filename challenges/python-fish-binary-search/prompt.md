## Прятки с сосиской (Binary Search) 🌭🔍

> 🏢 **Эта задача часто встречается на собеседованиях в:** **SpaceX**, **Google**, **Yandex**, **Сбер**, **Т-Банк**.
> 
> 😺 **ЛитКот напоминает:** Бинарный поиск требует строго отсортированного входного массива и работает путем деления пространства поиска пополам на каждом шаге.

Хозяин спрятал сосиску в одном из контейнеров, которые пронумерованы и упорядочены по возрастанию. Чтобы найти её за минимальное число шагов, вам нужно применить бинарный поиск.

### Задание:

Реализуйте функцию `find_sausage(containers, target)`, которая принимает отсортированный по возрастанию список чисел `containers` и искомое значение `target`. Функция должна возвращать индекс найденного элемента или `-1`, если элемент отсутствует в списке.

Ваше решение должно выполняться за время **O(log n)**.

```python
# Пример
find_sausage([1, 2, 3, 4, 5, 6, 7], 4) # 3
find_sausage([1, 2, 3, 4, 5, 6, 7], 10) # -1
```
