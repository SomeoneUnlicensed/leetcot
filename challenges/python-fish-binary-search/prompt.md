## Прятки с сосиской 🌭🔍

Мяу! Хозяин спрятал сосиску в одном из контейнеров, которые стоят в ряд и отсортированы по размеру. Чтобы не тратить время на проверку каждого контейнера, мы будем использовать бинарный поиск.

### Задание:
Реализуй функцию `find_sausage(containers, target)`, которая принимает отсортированный список чисел `containers` и искомое число `target`. Функция должна вернуть индекс элемента, если он найден, иначе `-1`.

Алгоритм должен работать за время **O(log n)**.

```python
# Пример
find_sausage([1, 2, 3, 4, 5, 6, 7], 4) # 3
find_sausage([1, 2, 3, 4, 5, 6, 7], 10) # -1
```
