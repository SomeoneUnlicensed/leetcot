## Найди самых голодных котов 🐱🍽️

Главный повар ЛитКота составляет список котов, которых нужно накормить в первую очередь. Критерий — суммарное количество рыбок за всё время.

Напишите `SELECT`, который вернёт **имена котов** и **суммарное количество рыбок** (`total_fish`), отсортированные по убыванию.

- `cats` (id, name)
- `meals` (id, cat_id, fish_count, meal_date)
