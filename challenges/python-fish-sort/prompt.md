## Сортировка улова 🐟

Мяу! Сегодня был удачный день на рыбалке. У нас есть список рыбок, и каждая имеет свой вес. Чтобы быть самым эффективным котом в округе, нам нужно отсортировать их **по убыванию** веса.

### Задание:
Реализуй функцию `sort_fish(fish_weights)`, которая принимает список чисел (веса рыбок) и возвращает новый список, отсортированный от большего к меньшему.

**Подсказка:** Используй быстрый алгоритм, например, QuickSort или встроенный метод Python, если хочешь просто пообедать.

```python
# Пример
sort_fish([5, 2, 9, 1, 5, 6]) # [9, 6, 5, 5, 2, 1]
```
