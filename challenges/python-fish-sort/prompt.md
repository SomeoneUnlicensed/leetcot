## Сортировка улова (Sort List) 🐟

> 🏢 **Эта задача часто встречается на собеседованиях в:** **SpaceX**, **Google**, **Yandex**, **Сбер**, **Т-Банк**.
> 
> 😺 **ЛитКот напоминает:** Сортировка — фундаментальная операция. Стремитесь использовать алгоритмы со сложностью $O(N \log N)$, такие как QuickSort или MergeSort, либо стандартные оптимизированные механизмы языка.

У вас есть список весов пойманных рыбок. Чтобы эффективно распределить добычу, вам нужно отсортировать их **по убыванию** веса.

### Задание:

Реализуйте функцию `sort_fish(fish_weights)`, которая принимает список чисел (веса рыбок) и возвращает новый список, отсортированный от большего к меньшему.

```python
# Пример
sort_fish([5, 2, 9, 1, 5, 6]) # [9, 6, 5, 5, 2, 1]
```
