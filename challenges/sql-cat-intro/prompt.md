## Добро пожаловать в «Рыбную Базу Данных» 🐱🗄️

Привет! Перед тобой — **интерактивный SQL-курс ЛитКота**, где вместо скучных теоретических примеров тебя ждут настоящие задачи с базами данных котиков, рыбками и тайными складами.

---

### 🎯 Что тебя ждёт

В каждом задании ты получишь:

- **Описание ситуации** — что сломалось, что нужно найти, что переложить
- **Реальную базу данных** — уже поднятую прямо в браузере (SQLite WASM, никакой установки!)
- **Таймер** — от 90 секунд до 2 минут. Успей!
- **Кота-помощника** — он следит за твоими запросами и радуется когда ты справляешься

---

### 🖥️ Как пользоваться терминалом

В правой панели находится **SQL-терминал**. Вот основные команды:

| Команда               | Что делает                                |
| --------------------- | ----------------------------------------- |
| `SELECT * FROM cats;` | Обычный SQL-запрос                        |
| `.schema`             | Показать структуру всех таблиц            |
| `.check`              | Проверить правильность твоего решения     |
| `.reset`              | Сбросить базу данных в исходное состояние |
| `↑ / ↓`               | История предыдущих запросов               |

---

### 📊 Типы задач

- **SELECT** — напиши запрос, который вернёт нужные данные
- **UPDATE / INSERT / DELETE** — измени состояние базы так, чтобы данные соответствовали условию

---

### 🐾 Правила курса

1. База данных **изолирована в твоём браузере** — никто не пострадает если ты всё сломаешь
2. Используй `.reset` чтобы начать задачу заново
3. Кот в углу — **индикатор настроения**: зелёный = всё хорошо, красный = ошибка
4. Задачи идут по нарастающей сложности

---

> 💡 **Совет:** Всегда начинай с `.schema` чтобы понять структуру таблиц. Это экономит время!

Когда будешь готов — нажми **«Отметить как выполненное»** и переходи к первому испытанию. Удачи! 🐟
