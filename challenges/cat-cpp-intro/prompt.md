## Кошачья структура на C++ 🐱💻

Добро пожаловать в мир C++ и его мощной Standard Template Library (STL)! Сегодня мы создадим кошачью структуру данных, которая будет использовать основные компоненты STL.

### Основы C++

C++ - это мощный язык, который позволяет:
1. **Работать с памятью**: управление указателями и ссылками.
2. **Использовать STL**: контейнеры (`vector`, `map`), алгоритмы и итераторы.
3. **Объектно-ориентированное программирование**: классы, наследование, полиморфизм.

### Задание:

Создай класс `Cat` со следующими методами:
- Конструктор `Cat(string name, int age)`: инициализирует кота с именем и возрастом.
- `string getName()`: возвращает имя кота.
- `int getAge()`: возвращает возраст кота.
- `void greet()`: выводит приветствие в формате "Мяу! Я {name}, мне {age} лет!"

Используй `std::string` для работы со строками.
