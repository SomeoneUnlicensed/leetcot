## Кот Kotlin: Лаконичность на JVM 🐱

Kotlin - это современный язык на JVM, который избегает излишней писанины Java.

### Основы Kotlin

1. **Data classes**: Специальный синтаксис для класса с данными
2. **Методы**: Простое определение
3. **Строковые шаблоны**: Встроенная интеграция переменных в строки

### Задача

Создайте класс `Cat` с:
- Конструктором, принимающим `name` (String)
- Методом `meow()`, возвращающим "Мяу, $name!"
- Методом `greet(other: String)`, возвращающим "$name говорит привет $other!"
