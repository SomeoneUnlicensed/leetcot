## Параллельный Тыгыдык: Очередь Кормления 🐹💨

Мяу! Наступило время обеда, и стая из сотни котов устроила **Параллельный Тыгыдык**. Чтобы никто не остался голодным и не подрался, нам нужно реализовать эффективную систему распределения еды, используя мощь **Go Channels** и **Goroutines**.

### Конкурентность в Go

Go позволяет выполнять задачи параллельно с помощью:
1. **Goroutines**: легковесные потоки выполнения.
2. **Channels**: способ безопасной передачи данных между горутинами.
3. **WaitGroups**: способ дождаться завершения всех параллельных задач.

### Задание:
Реализуй функцию `FeedCats(catNames []string, workersCount int) map[string]bool`.
- `catNames`: список имен голодных котов.
- `workersCount`: количество мисок (воркеров), которые могут выдавать еду одновременно.

Каждый кот должен быть "накормлен" в отдельной горутине. Функция должна вернуть карту, где ключ — имя кота, а значение — `true` (накормлен).
Используй канал для передачи имен котов воркерам.
