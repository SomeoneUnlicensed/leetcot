## Объединение рыбных зон 🐟📅

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **SpaceX**, **Yandex**, **Сбер**, **Т-Банк**.
> 
> 😺 **ЛитКот напоминает:** Работа с интервалами времени требует аккуратности при сравнении границ. Убедитесь, что список отсортирован перед объединением.

Вам дан список интервалов времени `intervals`, где каждый интервал представляет собой пару `[start, end]`, в течение которого коты могут ловить рыбу в определенной зоне.

Некоторые зоны перекрываются по времени. Ваша задача — объединить все перекрывающиеся интервалы и вернуть новый список непересекающихся интервалов, которые полностью покрывают все исходные интервалы.

### Задание:

Реализуйте функцию `merge_intervals(intervals)`, которая принимает список интервалов и возвращает список объединенных интервалов.

```python
# Пример
merge_intervals([[1, 3], [2, 6], [8, 10], [15, 18]])
# Результат: [[1, 6], [8, 10], [15, 18]]
# Объяснение: Интервалы [1, 3] и [2, 6] перекрываются, поэтому они объединяются в [1, 6].
```
