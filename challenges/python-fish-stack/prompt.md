## Скобочный Тыгыдык 🐈💨

Мяу! Кот устроил ночной тыгыдык и перепутал в шкафу все открытые и закрытые коробки. Нам нужно проверить, все ли коробки закрыты правильно.

### Задание:
Реализуй функцию `is_valid_boxes(s)`, которая принимает строку из символов `()`, `[]`, `{}` и возвращает `True`, если все скобки сбалансированы и закрыты в правильном порядке, иначе `False`.

```python
# Пример
is_valid_boxes("()[]{}") # True
is_valid_boxes("(]")     # False
is_valid_boxes("([)]")   # False
```
