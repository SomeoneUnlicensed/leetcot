## Скобочный Тыгыдык (Valid Parentheses) 🐈💨

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **Apple**, **Yandex**, **Сбер**, **Т-Банк**.
>
> 😺 **ЛитКот напоминает:** Стек (LIFO — Last In, First Out) идеально подходит для проверки вложенных и парных структур, так как последняя открытая коробка/скобка должна закрываться первой.

Кот устроил ночной тыгыдык и перепутал в шкафу все открытые и закрытые коробки. Вам необходимо проверить, правильно ли сбалансированы и закрыты все коробки (скобки).

### Задание:

Реализуйте функцию `is_valid_boxes(s)`, которая принимает строку, состоящую из символов `()`, `[]`, `{}` и возвращает `True`, если все скобки сбалансированы и закрыты в правильном порядке, иначе `False`.

Строка считается валидной, если:

1. Открытые скобки закрываются скобками того же типа.
2. Открытые скобки закрываются в правильном порядке.
3. Каждой закрывающей скобке соответствует открывающая скобка того же типа.

```python
# Пример
is_valid_boxes("()[]{}") # True
is_valid_boxes("(]")     # False
is_valid_boxes("([)]")   # False
```
