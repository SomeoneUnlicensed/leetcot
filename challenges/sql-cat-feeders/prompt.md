## Починить миски котов (UPDATE) 🥣🔧

База данных автокормушек ЛитКота начала барахлить. Кормушки Мурзика и Барсика выдают ошибку и имеют статус `'broken'`.

Напишите `UPDATE`-запрос, который изменит статус сломанных кормушек на `'active'` для всех кормушек, принадлежащих котам с именами `'Мурзик'` и `'Барсик'`.

### Схема таблиц:

- `cats` (id, name)
- `feeders` (id, cat_id, name, status)

```sql
-- До:  status = 'broken'
-- После: status = 'active'
```
