## Рейтинг котов-рыбаков 🎣🏆

ЛитКот проводит ежегодный турнир рыбаков. Составьте рейтинговую таблицу — для каждого кота посчитайте количество выходов на рыбалку и средний улов. Отберите только тех, кто ловил **не менее 3 раз**, и отсортируйте по среднему улову по убыванию.

### Схема таблиц:

- `cats` (id, name)
- `catches` (id, cat_id, fish_count, caught_at)

```sql
-- Ожидаемые столбцы: name | times_fished | avg_catch
-- avg_catch округлить до 1 знака
```
