## Кот Rust: Владение памятью 🦀

Rust требует безопасности памяти, и мы изучим это на примере кошачьего приложения.

### Концепции Rust

1. **Владение (Ownership)**: Каждое значение имеет одного хозяина
2. **Заимствование (Borrowing)**: Временное использование без передачи владения
3. **Время жизни (Lifetime)**: Как долго действителен кот

### Задача

Создайте структуру `Cat` с методами:
- `new(name: String) -> Cat` - конструктор
- `meow(&self) -> String` - выдает "Мяу, {name}!"
- `greet(&self, other: &str) -> String` - выдает "{name} говорит привет {other}!"
