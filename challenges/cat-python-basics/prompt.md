### Списки и Циклы

В Python мы храним вещи в списках `[]`. Например, список имен твоих друзей:
```python
cats = ["Барсик", "Мурзик", "Рыжик"]
```

Чтобы пройтись по каждому котику, мы используем цикл `for`:
```python
for cat in cats:
    print(f"Привет, {cat}!")
```
