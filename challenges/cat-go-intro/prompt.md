## Быстрый как Кот: Введение в Go 🐹🐾

Мяу! Go — это язык для тех, кто хочет бегать быстро, как кот за лазерной указкой. Он строгий, надежный и компилируется в один маленький файл.

### Структуры (Structs)

В Go нет классов в привычном понимании, но есть **структуры**. Это как карточка кота в приюте:
```go
type Cat struct {
    Name  string
    Age   int
    Power float64
}
```

Чтобы создать нового кота:
```go
barsik := Cat{Name: "Барсик", Age: 5, Power: 9000.1}
```
