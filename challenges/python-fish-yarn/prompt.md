## Распутывание Клубка 🧶

Мяу! Кот запутался в клубке ниток, который представляет собой связный список. Чтобы освободиться, нам нужно развернуть этот список в обратном порядке.

### Задание:
Реализуй функцию `reverse_yarn(head)`, которая принимает голову связного списка и возвращает голову развернутого списка.

Узел списка определен так:
```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

```python
# Пример
# 1 -> 2 -> 3 -> 4 -> 5
# Результат: 5 -> 4 -> 3 -> 2 -> 1
```
