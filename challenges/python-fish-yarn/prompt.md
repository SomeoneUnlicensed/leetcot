## Распутывание Клубка (Reverse Linked List) 🧶

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **Apple**, **Yandex**, **Сбер**, **Т-Банк**.
> 
> 😺 **ЛитКот напоминает:** Эту задачу можно решить как итеративно, так и рекурсивно. Попробуйте итеративный подход с перенаправлением указателей для экономии памяти ($O(1)$ по памяти).

Клубок ниток представляет собой односвязный список. Чтобы распутать его, вам нужно развернуть этот список в обратном порядке.

### Задание:

Реализуйте функцию `reverse_yarn(head)`, которая принимает голову связного списка и возвращает голову развернутого списка.

Узел списка определен следующим образом:

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next
```

```python
# Пример
# Вход:  1 -> 2 -> 3 -> 4 -> 5
# Выход: 5 -> 4 -> 3 -> 2 -> 1
```
