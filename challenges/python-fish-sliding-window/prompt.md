## Лазерная указка 🔦🔴

Мяу! Хозяин водит лазерной указкой по стене, оставляя след из разных «точек» (символов). Кот хочет поймать самую длинную последовательность точек, в которой **нет повторяющихся** символов.

### Задание:
Реализуй функцию `longest_laser_track(s)`, которая принимает строку и возвращает длину самой длинной подстроки без повторяющихся символов.

```python
# Пример
longest_laser_track("abcabcbb") # 3 ("abc")
longest_laser_track("bbbbb")    # 1 ("b")
longest_laser_track("pwwkew")   # 3 ("wke")
```
