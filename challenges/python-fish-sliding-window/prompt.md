## Лазерная указка (Longest Substring Without Repeating Characters) 🔦🔴

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **Apple**, **Yandex**, **Сбер**, **Т-Банк**.
>
> 😺 **ЛитКот напоминает:** Метод «скользящего окна» (Sliding Window) с использованием двух указателей и хэш-сета (множества) позволяет решить задачу за линейное время $O(N)$.

Лазерный луч оставляет след из символов. Вам необходимо найти длину самой длинной непрерывной последовательности символов, в которой нет повторяющихся знаков.

### Задание:

Реализуйте функцию `longest_laser_track(s)`, которая принимает строку и возвращает длину самой длинной подстроки без повторяющихся символов.

```python
# Пример
longest_laser_track("abcabcbb") # 3 ("abc")
longest_laser_track("bbbbb")    # 1 ("b")
longest_laser_track("pwwkew")   # 3 ("wke")
```
