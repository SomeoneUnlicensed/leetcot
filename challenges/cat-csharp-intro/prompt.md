## Кот C#: Объектная ориентация на .NET 🐱

C# - это мощный язык на платформе .NET. Давайте создадим класс Cat с поддержкой C#.

### Основы C#

1. **Классы**: Основные строительные блоки
2. **Свойства (Properties)**: Безопасный доступ к данным
3. **Методы**: Действия объекта

### Задача

Создайте класс `Cat` с:
- Конструктором, принимающим `name` (string)
- Методом `Meow()`, возвращающим "Мяу, {name}!"
- Методом `Greet(string other)`, возвращающим "{name} говорит привет {other}!"
