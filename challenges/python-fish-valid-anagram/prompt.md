## Рыбные анаграммы 🔠🐱

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **Yandex**, **Т-Банк**, **Apple**, **Сбер**.
> 
> 😺 **ЛитКот напоминает:** Две строки являются анаграммами, если они состоят из абсолютно одинакового набора символов с одинаковой частотой их вхождения.

Вам даны две строки `s` и `t`. Нам нужно определить, является ли строка `t` анаграммой строки `s`.

### Задание:

Реализуйте функцию `is_anagram(s, t)`, которая возвращает `True`, если строки являются анаграммами, и `False` в противном случае.

```python
# Пример
is_anagram("cat", "tac")      # True
is_anagram("rat", "car")      # False
```
