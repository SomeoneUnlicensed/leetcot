## Очередь за кормом 🥣🐈

Мяу! Наступило время обеда. Коты выстроились в очередь, и у каждого есть свой уровень голода. Мы должны кормить самого голодного кота первым. Если уровни голода одинаковы, кормим любого из них.

### Задание:
Реализуй класс `FeedingQueue`, у которого есть два метода:
1. `add_cat(hunger_level)`: добавляет кота с определенным уровнем голода.
2. `feed_next()`: возвращает уровень голода самого голодного кота и удаляет его из очереди. Если очередь пуста, верни `None`.

**Подсказка:** Используй кучу (heap), чтобы операция `feed_next` работала максимально быстро. В Python `heapq` реализует min-heap, поэтому для max-heap используй отрицательные значения.

```python
# Пример
q = FeedingQueue()
q.add_cat(5)
q.add_cat(10)
q.add_cat(3)
q.feed_next() # 10
q.feed_next() # 5
```
