## Очередь за кормом (Max-Priority Queue) 🥣🐈

> 🏢 **Эта задача часто встречается на собеседованиях в:** **SpaceX**, **Google**, **Apple**, **Yandex**.
> 
> 😺 **ЛитКот напоминает:** Очередь с приоритетом строится на базе двоичной кучи (binary heap). В Python модуль `heapq` реализует min-heap. Чтобы получить max-heap, умножайте добавляемые числовые значения на `-1`.

Наступило время обеда. Коты выстроились в очередь, и у каждого есть свой уровень голода. Вам необходимо кормить самого голодного кота первым. Если уровни голода одинаковы, допускается покормить любого из них.

### Задание:

Реализуйте класс `FeedingQueue`, предоставляющий два метода:

1. `add_cat(hunger_level)`: регистрирует кота с определенным уровнем голода (целое число).
2. `feed_next()`: возвращает уровень голода наиболее голодного кота и извлекает его из очереди. Если очередь пуста, метод должен возвращать `None`.

Операции добавления и извлечения должны работать за время **O(log n)**.

```python
# Пример использования
q = FeedingQueue()
q.add_cat(5)
q.add_cat(10)
q.add_cat(3)
q.feed_next() # 10
q.feed_next() # 5
```
