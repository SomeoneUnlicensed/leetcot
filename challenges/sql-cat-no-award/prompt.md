## Коты без наград 🏆🚫

Главный кот ЛитКота хочет знать, кто из котов ещё ни разу не получал награду. Такие коты получат дополнительную порцию рыбы.

Найдите всех котов, которых **нет в таблице `awards`**.

### Схема таблиц:

- `cats` (id, name)
- `awards` (id, cat_id, award_name)

-- Ожидаемые столбцы: id | name
