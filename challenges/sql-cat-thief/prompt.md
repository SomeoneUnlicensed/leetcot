## Похититель сосисок (JOIN & LIMIT) 🌭🕵️‍♂️

😺 **ЛитКот напоминает:** Кто-то коварно утащил сосиску со стола! У нас есть журнал логов действий котиков `action_logs` и реестр котиков `cats`.

Нам нужно объединить таблицы и узнать **имя кота** (столбец `name`), у которого последнее по времени (`created_at`) действие (`action`) было `'stole_sausage'`.

Напишите `SELECT` запрос, возвращающий одну строку с единственной колонкой `name`.

### Схема таблиц:

- `cats` (id, name, color)
- `action_logs` (id, cat_id, action, created_at)

-- Ожидается результат в виде одной ячейки:
-- name
