## Золотая рыбка 👑🐟

Мяу! На другом конце города в пруду плавает золотая рыбка. Дорога туда лежит через дворы, в каждом из которых живут злые собаки. Каждая дорога имеет свою «стоимость» в единицах кошачьей энергии. Нам нужно найти путь, который потребует **минимум энергии**.

### Задание:
Реализуй функцию `shortest_energy_path(graph, start, target)`, которая принимает граф в виде словаря смежности с весами и находит минимальную энергию для перехода от `start` до `target`. Если пути нет, верни `float('inf')`.

```python
# Пример графа
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}
shortest_energy_path(graph, 'A', 'D') # 4 (A -> B -> C -> D)
```
