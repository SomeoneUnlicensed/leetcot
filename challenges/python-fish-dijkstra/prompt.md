## Золотая рыбка (Dijkstra's Algorithm) 👑🐟

> 🏢 **Эта задача часто встречается на собеседованиях в:** **SpaceX**, **Google**, **Yandex**.
>
> 😺 **ЛитКот напоминает:** Алгоритм Дейкстры находит кратчайшие пути от одной из вершин графа до всех остальных во взвешенном графе с неотрицательными весами рёбер. Для эффективного выбора следующего узла с минимальным расстоянием используйте приоритетную очередь (кучу).

В пруду на другом конце города плавает золотая рыбка. Дорога к ней лежит через дворы с препятствиями. Каждая дорога имеет свою «стоимость» в единицах кошачьей энергии. Вам нужно найти путь, который потребует **минимального количества энергии**.

### Задание:

Реализуйте функцию `shortest_energy_path(graph, start, target)`, которая принимает граф в виде словаря смежности с весами рёбер и находит минимальную суммарную энергию для перехода от узла `start` до узла `target`. Если пути не существует, верните `float('inf')`.

```python
# Пример графа
graph = {
    'A': {'B': 1, 'C': 4},
    'B': {'A': 1, 'C': 2, 'D': 5},
    'C': {'A': 4, 'B': 2, 'D': 1},
    'D': {'B': 5, 'C': 1}
}
shortest_energy_path(graph, 'A', 'D') # 4 (A -> B -> C -> D)
```
