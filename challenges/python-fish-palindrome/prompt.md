## Кот в Зеркале 🐈🪞

Мяу! Кот смотрит в зеркало и видит точно такого же кота. Нам нужно проверить, является ли строка палиндромом (читается одинаково в обе стороны), игнорируя регистр и небуквенные символы.

### Задание:
Реализуй функцию `is_palindrome_cat(s)`, которая возвращает `True`, если строка является палиндромом, иначе `False`.

```python
# Пример
is_palindrome_cat("A man, a plan, a canal: Panama") # True
is_palindrome_cat("race a car") # False
```
