## Утилита Выбора (Pick)

Мяу! Представь, что у нас есть огромный шкаф с кошачьими игрушками, но нам нужно выбрать только мышку и мячик. В TypeScript это называется `Pick`.

### Как это работает?
Допустим, у нас есть тип "Полный Кот":
```ts
interface FullCat {
  name: string;
  age: number;
  lives: number;
  favoriteFood: string;
}
```

Но для простой прогулки нам нужно только имя и возраст. Мы можем создать новый тип с помощью нашего `MyPick`:
```ts
type WalkingCat = MyPick<FullCat, "name" | "age">;
```

### Задание:
Реализуй тип `MyPick<T, K>`. Он должен принимать исходный тип `T` и набор ключей `K`, которые нужно оставить. Помни, что `K` должен быть подмножеством ключей `T`!
