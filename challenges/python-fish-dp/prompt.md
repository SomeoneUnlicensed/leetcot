## Максимум сосисок 🌭

Мяу! На столе разложены ряды сосисок с разной питательностью. Кот может прыгать по столу, но есть правило: если он съел сосиску в одной позиции, он не может есть сосиску в соседней позиции (слишком много улик!).

### Задание:
Реализуй функцию `max_sausages(sausages)`, которая принимает список целых чисел (ценность сосисок) и возвращает максимально возможную сумму, которую кот может собрать, не выбирая две соседние сосиски.

```python
# Пример
max_sausages([1, 2, 3, 1]) # 4 (1 + 3)
max_sausages([2, 7, 9, 3, 1]) # 12 (2 + 9 + 1)
```
