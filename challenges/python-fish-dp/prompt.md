## Максимум сосисок (House Robber) 🌭

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **Yandex**, **Сбер**.
> 
> 😺 **ЛитКот напоминает:** Это классическая задача динамического программирования. Для каждой сосиски определите оптимальный выбор: взять ее (прибавив ценность к оптимальному результату для шага $i-2$) или пропустить (сохранив результат для шага $i-1$).

На столе разложен ряд сосисок с разной степенью питательности. Кот может прыгать по столу, но существует правило конспирации: если он съел сосиску в некоторой позиции, он не может есть сосиску в непосредственно соседней позиции.

### Задание:

Реализуйте функцию `max_sausages(sausages)`, которая принимает список неотрицательных целых чисел (ценность сосисок) и возвращает максимально возможную сумму, которую можно собрать, не выбирая две соседние сосиски.

Решение должно работать за линейное время $O(N)$ и использовать $O(1)$ дополнительной памяти.

```python
# Пример
max_sausages([1, 2, 3, 1]) # 4 (1 + 3)
max_sausages([2, 7, 9, 3, 1]) # 12 (2 + 9 + 1)
```
