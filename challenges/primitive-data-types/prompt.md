## Примитивные типы данных (Основы ЛитКота)

Мяу! В мире ЛитКота всё начинается с простых вещей. Чтобы стать настоящим мастером TypeScript, ты должен понимать, из чего состоит каждый котик.

### 1. Строки (`string`)
Это всё, что касается текста. Имя кота, его порода или звук, который он издает, когда ты наступил ему на хвост.

```ts
const catName: string = "Барсик";
```

### 2. Числа (`number`)
Возраст кота, количество жизней (обычно 9) или вес в килограммах. В TypeScript нет разницы между целыми числами и дробными — всё это `number`.

```ts
const catAge: number = 3;
```

### 3. Булевы значения (`boolean`)
Кот либо голоден, либо... нет, погоди, коты всегда голодны. Но технически это `true` (истина) или `false` (ложь).

```ts
const isHungry: boolean = true;
```
