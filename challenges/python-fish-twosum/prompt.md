## Пара рыбок 🐟🐟

Мяу! Нам нужно выбрать ровно две рыбки из списка так, чтобы их суммарный вес был равен заданному числу `target`.

### Задание:
Реализуй функцию `two_fish(weights, target)`, которая возвращает индексы двух чисел из списка `weights`, сумма которых равна `target`. 
Каждый вход имеет ровно одно решение, и нельзя использовать один и тот же элемент дважды.

```python
# Пример
two_fish([2, 7, 11, 15], 9) # [0, 1]
```
