## Пара рыбок (Two Sum) 🐟🐟

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **Apple**, **Yandex**, **Сбер**, **Т-Банк**.
>
> 😺 **ЛитКот напоминает:** Оптимальное решение работает за линейное время $O(N)$ с использованием хэш-таблицы (словари в Python).

Вам необходимо выбрать ровно две рыбки из списка так, чтобы их суммарный вес был равен заданному числу `target`.

### Задание:

Реализуйте функцию `two_fish(weights, target)`, которая возвращает индексы двух чисел из списка `weights`, сумма которых равна `target`.

Каждый набор входных данных имеет ровно одно решение. Не допускается использовать один и тот же элемент дважды. Вы можете возвращать индексы в любом порядке.

```python
# Пример
two_fish([2, 7, 11, 15], 9) # [0, 1]
```
