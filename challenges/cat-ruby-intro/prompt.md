## Кот Ruby: Элегантный код 💎

Ruby известен своей красотой и простотой. Давайте создадим кошачий класс.

### Основы Ruby

1. **Классы**: Определяются как `class Cat ... end`
2. **Методы инициализации**: `def initialize(name)`
3. **Методы экземпляра**: Действия для кота

### Задача

Создайте класс `Cat` с:
- Конструктором `initialize(name)`
- Методом `meow`, возвращающим "Мяу, {name}!"
- Методом `greet(other)`, возвращающим "{name} говорит привет {other}!"
