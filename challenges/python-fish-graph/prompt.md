## Путь к Миске (Breadth-First Search) 🐾

> 🏢 **Эта задача часто встречается на собеседованиях в:** **Google**, **Yandex**, **Сбер**, **Т-Банк**.
>
> 😺 **ЛитКот напоминает:** Поиск в ширину (BFS) использует очередь (FIFO) и идеально подходит для нахождения кратчайшего пути в невзвешенном графе.

Кот голоден, а миска находится в другом конце квартиры. Квартира представлена в виде графа, где узлы — это комнаты, а рёбра — открытые двери между ними.

### Задание:

Реализуйте функцию `find_shortest_path(graph, start, end)`, которая находит длину кратчайшего пути (количество переходов между комнатами) от комнаты `start` до комнаты `end`. Если пути не существует, верните `-1`.

Граф передается в виде словаря смежности:

```python
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    # ...
}
```

```python
# Пример
find_shortest_path(graph, 'A', 'F') # 2 (A -> C -> F)
```
