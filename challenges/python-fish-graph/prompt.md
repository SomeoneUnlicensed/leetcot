## Путь к Миске (Поиск в ширину) 🐾

Мяу! Кот очень голоден, а миска находится в другом конце квартиры. Квартира представлена в виде графа, где узлы — это комнаты, а ребра — двери между ними.

### Задание:
Реализуй функцию `find_shortest_path(graph, start, end)`, которая находит длину кратчайшего пути от комнаты `start` до комнаты `end`. Если пути нет, верни `-1`.

Граф передается в виде словаря смежности:
```python
graph = {
    'A': ['B', 'C'],
    'B': ['A', 'D', 'E'],
    'C': ['A', 'F'],
    # ...
}
```

```python
# Пример
find_shortest_path(graph, 'A', 'F') # 2 (A -> C -> F)
```
