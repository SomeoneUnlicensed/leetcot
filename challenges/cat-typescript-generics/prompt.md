## Кот TypeScript: Дженерики 🐱

Дженерики позволяют нам создавать гибкие и повторно используемые типы. Давайте создадим кошачий контейнер.

### Основы Дженериков

1. **Параметры типа**: `<T>` - позволяет использовать любой тип
2. **Ограничения**: `<T extends string>` - ограничивает возможные типы
3. **Множественные параметры**: `<T, U>` - использование нескольких типов

### Задача

Создайте тип-функцию `CatContainer<T>` которая:
- Хранит значение типа T
- Имеет метод `get(): T`
- Имеет метод `set(value: T): void`

### Пример

```typescript
const container = new CatContainer<string>("Барсик");
console.log(container.get()); // "Барсик"
```
