## Мяу на Java: Объектно-ориентированное мышление 🐱

Java - это язык, где всё вращается вокруг **объектов**. Давайте создадим класс `Cat`, который будет представлять нашего кота.

### Основы Java

1. **Классы и объекты**: Java позволяет нам создавать классы для описания сущностей
2. **Атрибуты**: Свойства, которые есть у объекта (например, имя кота)
3. **Методы**: Действия, которые может выполнять объект

### Задача

Создайте класс `Cat` с:
- Конструктором, принимающим `name` (String)
- Методом `meow()`, который возвращает строку "Мяу, {name}!"
- Методом `greet(String other)`, который возвращает "{name} говорит привет {other}!"

### Пример использования

```java
Cat cat = new Cat("Барсик");
System.out.println(cat.meow()); // Мяу, Барсик!
System.out.println(cat.greet("Пушок")); // Барсик говорит привет Пушок!
```
